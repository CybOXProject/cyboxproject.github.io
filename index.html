Schemas

Development of the CybOX schemas occurs in the schemas repository.

If you're looking for official releases of the CybOX Language schemas, check out the main CybOX site.

Language Bindings

To generate or parse CybOX content within another program, you can use the libraries we've developed for other programming languages. Currently we only have language bindings for Python. See the python-cybox repository for more information.

Tools

Tools for converting between CybOX and various other formats can be found in the following repos:

  • CybOX to HTML (XSLT transform)
  • CybOX to OVAL XML (Python script)
  • RFC 822 email to CybOX (Python script)
  • OpenIOC XML to CybOX (Python script)
  • X.509 certificate plain text to CybOX (Python script)

Contributing

We welcome any and all feedback related to CybOX and the various projects discussed above. If you have a GitHub account (or want to create one), you can create bug reports or features requests within the respective repository. Pull requests are especially appreciated. You can also send an email to the public cybox-discussion-list, or a private email to cybox@mitre.org.
